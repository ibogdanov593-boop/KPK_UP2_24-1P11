### Вариант №1. Сервис аутентификации (Auth)

#### Регистрация пользователя

Информация требуемая для создания пользователя:

| Параметр          | Обязательность | Тип       | Ограничение        | Значение по умолчанию |
|-------------------|----------------|-----------|--------------------|-----------------------|
| Имя пользователя  | Обязательно    | Строка    | Уникальное         |                       |
| Электронная почта | Обязательно    | Строка    | Уникальная         |                       |
| Пароль            | Обязательно    | Строка    | Должен быть не менее 8 символов |                 |

Комбинация параметров: Имя пользователя и Электронная почта должна быть уникальной.

#### Вход пользователя по логину и паролю

Входные параметры:

| Параметр         | Обязательность | Тип     | Ограничение          | Значение по умолчанию |
|------------------|----------------|---------|----------------------|-----------------------|
| Имя пользователя | Обязательно    | Строка  | Уникальное           |                       |
| Пароль           | Обязательно    | Строка  |                      |                       |

Выходные данные:

| Параметр         | Тип     | Описание                |
|------------------|---------|-------------------------|
| access_token     | Строка  | JWT токен для доступа   |
| refresh_token    | Строка  | Токен для обновления    |

#### Сброс пароля

Информация требуемая для сброса пароля:

| Параметр         | Обязательность | Тип     | Ограничение          | Значение по умолчанию |
|------------------|----------------|---------|----------------------|-----------------------|
| Электронная почта| Обязательно    | Строка  | Уникальная           |                       |

Выходные данные:

| Параметр         | Тип     | Описание                |
|------------------|---------|-------------------------|
| message          | Строка  | Сообщение о результатах сброса пароля |

#### Получить информацию о пользователе по ID

Информация о пользователе:

| Параметр         | Тип      | Описание                |
|------------------|----------|-------------------------|
| id               | Целое    | Уникальный ID пользователя |
| Имя пользователя | Строка   | Имя пользователя        |
| Электронная почта| Строка   | Электронная почта пользователя |
| Статус           | Boolean  | Статус активности пользователя (True/False) |
| Дата создания    | Дата     | Дата и время создания пользователя |

#### Обновить информацию о пользователе по ID

Входные параметры:

| Параметр         | Обязательность | Тип     | Ограничение          | Значение по умолчанию |
|------------------|----------------|---------|----------------------|-----------------------|
| id               | Обязательно    | Целое   | Уникальный ID пользователя |                 |
| Имя пользователя | Не обязательно | Строка  |                      |                       |
| Электронная почта| Не обязательно | Строка  |                      |                       |
| Статус           | Не обязательно | Boolean |                      | True                  |

Выходные данные:

| Параметр         | Тип      | Описание                |
|------------------|----------|-------------------------|
| id               | Целое    | Уникальный ID пользователя |
| Имя пользователя | Строка   | Имя пользователя        |
| Электронная почта| Строка   | Электронная почта       |
| Статус           | Boolean  | Статус активности пользователя |

#### Удаление пользователя по ID

Вернет **True**, если пользователь был удален, иначе **False**.

#### Информация о пользователях

Параметры запроса:

| Параметр         | Тип     | Описание                |
|------------------|---------|-------------------------|
| Номер страницы   | Целое   | Номер страницы для пагинации |
| Количество на странице | Целое | Количество пользователей на одной странице |

Результат содержит только активных пользователей.

Информация о пользователях для пользователя:

| Параметр         | Тип      | Описание                |
|------------------|----------|-------------------------|
| id               | Целое    | Уникальный ID пользователя |
| Имя пользователя | Строка   | Имя пользователя        |
| Электронная почта| Строка   | Электронная почта       |
