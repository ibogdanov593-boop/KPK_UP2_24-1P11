# Вариант №5
# Сервис управления отделениями/факультетами СПО (Department Service)

## Сущность: Department (Отделение/Факультет)

### 1. Информация для создания сущности

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|-----------------------|
| `name` | Да | str | длина ≤ 20 | — |
| `course` | Да | int | от 1 до 4 | — |
| `specialty_code` | Да | str | ровно 8 цифр | — |
| `student_count` | Нет | int | от 0 до 50 | 0 |
| `curator` | Нет | str | длина ≤ 100 | `None` |

**Уникальные комбинации параметров:**
- `name`

### 2. Информация, возвращаемая при успешном создании

| Параметр | Тип |
|----------|-----|
| `id` | int |
| `name` | str |
| `course` | int |
| `specialty_code` | str |
| `student_count` | int |
| `curator` | str или `None` |

---

## Изменить сущность по ID

### 3. Информация для изменения сущности

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|-----------------------|
| `name` | Нет | str | длина ≤ 20 | текущее значение |
| `course` | Нет | int | от 1 до 4 | текущее значение |
| `student_count` | Нет | int | от 0 до 50 | текущее значение |
| `curator` | Нет | str | длина ≤ 100 | текущее значение |

> **Примечание:** Поле `specialty_code` изменять нельзя.

### 4. Информация, возвращаемая при успешном изменении

| Параметр | Тип |
|----------|-----|
| `id` | int |
| `name` | str |
| `course` | int |
| `specialty_code` | str |
| `student_count` | int |
| `curator` | str или `None` |

---

## Удалить сущность по ID

**Возвращаемое значение:**
- `True` — если группа была успешно удалена
- `False` — если группа с указанным ID не найдена

## Получить список сущностей по заданным параметрам

### 6. Параметры для получения списка

| Параметр | Тип | Описание |
|----------|-----|-----------|
| `name` | str | Поиск по части названия группы |
| `course` | int | Фильтр по курсу (1-4) |
| `specialty_code` | str | Точное совпадение кода специальности |
| `curator` | str | Поиск по ФИО куратора |
| `min_students` | int | Минимальное количество студентов |
| `max_students` | int | Максимальное количество студентов |
| `limit` | int | Максимум записей (по умолчанию 100) |
| `offset` | int | Пагинация (по умолчанию 0) |

### 7. Возвращаемый список сущностей

| Параметр | Тип |
|----------|-----|
| `id` | int |
| `name` | str |
| `course` | int |
| `specialty_code` | str |
| `student_count` | int |
| `curator` | str или `None` |
